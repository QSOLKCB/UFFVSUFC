# Spectral Crimes of Pangis Cosmology – Analysis Pack

This directory contains the written and support assets for the teardown of Pangis' “Unified Field Continuity” (UFC) cosmology.

## Contents

- `pangis_spectral_crimes.md`  
  Canonical Markdown source for the section **“Spectral Audit of Pangis ‘Unified Field Continuity’ Cosmology”**.

- `infographic_brief_spectral_crimes.md`  
  Design brief for the infographic **“Spectral Crimes of Pangis Cosmology”**.

- `pangis_param_comparison.csv`  
  Parameter comparison between UFC and standard multi-probe cosmology.

- `scripts/`  
  Small Python stubs for demonstrating how easy it is to fit log-periodic cosplay to noisy data.

Additional optional assets (not necessarily under version control):

- `pangis_spectral_crimes.pdf` – PDF export of `pangis_spectral_crimes.md`.  
- `spectral_crimes_of_pangis_cosmology.png` – final infographic export.

Use this folder as the source of truth for papers, NotebookLM projects, and visual assets related to the spectral audit.
